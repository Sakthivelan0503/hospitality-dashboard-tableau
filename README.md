# 📊 Hospitality Analytics Dashboard (Tableau)

## 📌 Project Overview

This project focuses on analyzing hospitality data to evaluate hotel performance and generate actionable insights.
The Tableau dashboard visualizes key business metrics such as occupancy rate, Average Daily Rate (ADR), and Revenue Per Available Room (RevPAR) to support data-driven decision-making.

---

## 🛠 Tools & Technologies

* Tableau
* Excel / CSV
* Data Visualization

---

## 📂 Dataset

The dataset includes hotel booking and revenue data with features such as:

* Booking details
* Occupancy rate
* Room revenue
* Customer trends
* Seasonal performance

---

## 📈 Key Features

* Interactive dashboard with filters, parameters, and drill-down functionality
* KPI tracking: Occupancy Rate, ADR, RevPAR
* Trend analysis for bookings and revenue
* City-wise and time-based performance insights
